マインドマップの開始と終了

マインドマップ全体を開始するには、@startmindmapと@endmindmapを使用します。これらのキーワードで図の始まりと終わりを示します。

    
@startmindmap
... // マインドマップの内容
@endmindmap

トピックを作成する

*, +, -, などの記号を使ってトピックを作成します。

            
@startmindmap
* OS
@endmindmap

枝を追加する

記号の数を増やして枝を追加できます。Markdownの構文で書くことも可能です。

            
@startmindmap
** Ubuntu
*** Linux Mint
*** Kubuntu 

 * Windows NT
  * Windows 8
  * Windows 10
@endmindmap

枝の方向を決める

left side キーワードで左側にトピックを配置できます。 

            
@startmindmap
* OS
** Ubuntu
*** Linux Mint
*** Kubuntu 

left side
 * Windows NT
  * Windows 8
  * Windows 10
@endmindmap

+ でトピックを作成した場合は、- で作成されたトピックが左側に作成されます。 

            
@startmindmap
+ OS
++ Ubuntu
+++ Linux Mint
+++ Kubuntu 

-- Windows NT
--- Windows 8
--- Windows 10
@endmindmap

マインドマップの構文と機能の詳細はPlantUMLの公式ドキュメンテーションを参照してください。
