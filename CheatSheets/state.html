状態図の開始と終了

状態図全体を開始するには、@startumlと@endumlを使用します。これらのキーワードで図の始まりと終わりを示します。

    
@startuml
... // 状態図の内容
@enduml

基本的な例

状態図の始点と終点は[*]で示します。矢印は --> で示します。

            
@startuml
[*] --> Standby : System Start
Standby --> Active : Turn On
Active --> Standby : Turn Off
Standby --> [*] : System Stop
@enduml

状態を入れ子にする

state キーワードと中括弧を使用して、状態を入れ子にすることができます。

            
@startuml
state Ready {
    [*] --> Standby : System Start
    Standby --> Active : Turn On
    Active --> Standby : Turn Off
    Standby --> [*] : System Stop
}
@enduml

状態図の構文と機能の詳細はPlantUMLの公式ドキュメンテーションを参照してください。
