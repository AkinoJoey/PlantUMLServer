アクティビティ図の開始と終了

アクティビティ図全体を開始するには、@startumlと@endumlを使用します。これらのキーワードで図の始まりと終わりを示します。

    
@startuml
... // アクティビティ図の内容
@enduml

アクティビティへのラベル付け

アクティビティのラベルは:で開始し;で終了します。

            
@startuml 
:システムを起動;
@enduml

図の開始と終了を示す

startキーワードで図の開始、stopキーワードまたはendキーワードで図の終了を示します。

            
@startuml
start
:システムを起動;
end
@enduml

条件文を追加

startキーワードで図の開始、stopキーワードまたはendキーワードで図の終了を示します。

            
@startuml
start
:システムを起動;

if(ゲストモードで参加しますか？)then(yes)
    :利用できるサービスは限定される
    履歴は残らない;
else(no)
    :履歴は残る;
endif

end
@enduml
